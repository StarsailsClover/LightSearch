## ✨ 核心功能速览

### 1. 多语言（5种）
```
设置 → 语言 → 选择你的语言
支持：中文、英语、日语、韩语、俄语
```

### 2. 主题（4种）
```
设置 → 个性化 → 主题
- LightSearch Classic（经典白）
- Classic Dark（深色）
- Liquid Glass（玻璃态）⭐ 推荐
- Eye Comfort（护眼绿）
```

### 3. Command Palette
```
按 Ctrl+K（Windows）或 Cmd+K（Mac）
快速访问所有功能
```

### 4. 学术搜索
```
点击右上角"学术搜索"
支持时间过滤和对比模式
```

### 5. 搜索对比
```
设置 → 高级 → 启用对比模式
在同一窗口对比多个搜索引擎
```

## 🎨 推荐配置

### 第一次使用建议

1. **切换到 Liquid Glass 主题**
   ```
   设置 → 个性化 → 主题 → Liquid Glass
   ```

2. **选择你的语言**
   ```
   设置 → 语言 → 选择
   ```

3. **添加常用搜索引擎**
   ```
   设置 → 搜索引擎 → 添加
   例如：https://www.google.com/search?q={query}
   ```

4. **尝试 Command Palette**
   ```
   按 Ctrl+K → 输入 "dark" → Enter
   （快速切换深色主题）
   ```

| `Enter` | 执行搜索 |
| `Esc` | 关闭弹窗 |
| `↑` / `↓` | 命令面板导航 |

## 🐛 遇到问题？

### 常见问题

**Q: 为什么搜索没反应？**
A: 检查是否启用了至少一个搜索引擎

**Q: Command Palette 不工作？**
A: 尝试刷新页面，或检查快捷键是否被占用

**Q: 主题切换后样式错乱？**
A: 按 Ctrl+F5 强制刷新
