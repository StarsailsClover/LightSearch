# 介绍

欢迎使用 **LightSearch** - 一个简洁优雅的聚合搜索引擎！

## 什么是 LightSearch？

LightSearch 是一个开源的聚合搜索工具，它允许你同时在多个搜索引擎中进行搜索，大大提高了信息检索的效率。

### 核心特性

- 🔍 **多引擎聚合** - 同时搜索 Google、Bing、百度等多个搜索引擎
- 🎓 **学术搜索** - 集成 Google Scholar、PubMed、arXiv 等学术资源
- 🎨 **主题系统** - 多款精美主题，支持深色模式
- ⚡ **零依赖** - 纯原生 JavaScript/CSS/HTML 实现
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🔧 **高度可定制** - 自定义搜索引擎和主题

## 为什么选择 LightSearch？

### 简洁高效

LightSearch 采用极简设计理念，界面清爽，操作直观。无需复杂配置，开箱即用。

### 隐私优先

所有数据都存储在本地，不会上传到任何服务器。你的搜索历史和配置完全私密。

### 开源免费

LightSearch 基于 GPL-3.0 许可证开源，完全免费使用，欢迎贡献代码。

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/StarsailsClover/LightSearch.git

# 进入目录
cd LightSearch

# 打开 index.html
open index.html
```
